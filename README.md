# ZeTheta Securitization Dashboard

## Overview

This project analyzes a securitized loan portfolio using Power BI. It enables interactive monitoring of portfolio performance, IFRS 9 stages, delinquency trends, and risk indicators.

## Objectives

- Portfolio performance monitoring
- IFRS 9 Stage analysis
- DPD bucket analysis
- Risk assessment
- Interactive dashboards

## Tools Used

- Power BI
- DAX
- Excel
- Power Query

## Dashboard Features

- Executive Summary
- Portfolio Risk Analysis
- IFRS Stage Analysis
- DPD Bucket Analysis
- Interactive Filters

## Key Metrics

- Total Exposure
- Outstanding Balance
- Delinquency Percentage
- Stage-wise Distribution
- Portfolio Risk

## Skills Demonstrated

- Data Cleaning
- Data Modeling
- DAX Measures
- Dashboard Design
- Business Intelligence
- Financial Analytics
